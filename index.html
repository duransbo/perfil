Ideologias

Performance acima de tudo

Nenhum usuário permanece muito tempo em uma aplicação lenta, ninguém gosta de esperar. Apesar de existir diversos outros fatores para que uma aplicação tenha uma boa usabilidade, levar em conta a performance como o pilar principal para o desenvolvimento trás diversas vantagens para o desenvolvimento, pois sempre obriga o desenvolvedor a melhora-la.

Código de qualidade

Um gancho da ideologia acima, pois não é possível atingir o máximo de performance com um código mal planejado. Código poluído, linhas desnecessárias, paradigma ineficiente e uma logica não funcional são características de uma aplicação que normalmente possui uma performance ruim, difícil de dar manutenção e escalar.
